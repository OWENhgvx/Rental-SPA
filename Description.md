# AirBrB – React Accommodation Platform

AirBrB 是一个基于 **React** 构建的住宿预订平台。  
项目以真实业务场景为模板，支持房源浏览、搜索、发布、预订、评论、房东管理等功能。

本项目重点展示了我在 **前端工程化、SPA 架构设计、组件封装、状态管理、REST API 集成、UI/UX 设计与自动化测试** 方面的能力。

---

## 🚀 技术亮点（个人能力展示）

### 🌐 前端架构设计
- 使用 **React Hooks + React Router** 构建完整 SPA  
- 全站无刷新路由切换、页面状态即时更新  
- 模块化组件结构、复用性高、代码可维护性强

### 🔗 REST API 集成
- 与后端接口完成全面联动  
- 实现登录/注册、房源管理、预订流程、评论系统等完整业务链路  
- 处理跨域、错误状态、异步加载与缓存逻辑

### 🧠 复杂业务逻辑处理
- 房源发布与多日期范围管理  
- 搜索过滤（标题/城市、卧室数、价格区间、评分、日期范围）  
- 订单状态流转（pending / accepted / rejected）  
- 房东收益统计、今年预订天数汇总、历史记录等

### 🎨 UI & 交互设计
- 组件化 UI（MUI + 自定义组件）  
- 表单校验、错误反馈、弹窗与通知系统  
- 列表、卡片、图表等完整页面布局

### 📊 数据可视化
- 绘制房东近 30 天收益趋势图  
- 动态渲染、自动更新

### 🔔 实时体验（轮询）
- 实现 Host/Guest 的预订通知系统  
- 未读标记、通知面板、交互细节完整

### 🧪 自动化测试
- **Component Testing**：对核心组件进行单元测试（props 行为、渲染逻辑）  
- **UI Testing（Cypress）**：覆盖完整“用户成功路径”  
- 使用 Jest 模拟 fetch、mock API 请求

## ✨ 功能概览

### 👤 用户系统
- 注册 / 登录 / 注销  
- 登录后可管理房源、预订、评论等

### 🏠 房东功能（Host）
- 创建 / 编辑 / 删除房源  
- 发布 / 下架房源  
- 设置多段可用日期  
- 查看预订请求并接受/拒绝  
- 查看今年收益、预订天数、历史记录  
- 支持视频缩略图（可选扩展）

### 🔍 房源浏览与搜索（Guest）
- 浏览所有已发布房源  
- 搜索：标题/城市、卧室数、价格、评分、日期范围  
- 进入详情页查看完整信息

### 📅 预订与评论
- 发起预订（可重复、可重叠）  
- 查看预订状态  
- 已接受的预订可留下评论  
- 评分按星级展示
